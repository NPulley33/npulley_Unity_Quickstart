# NPulley's Unity Quickstart Project

A (get this) quick start to any Unity project

## Do you hate setting up a new Unity project? Do you want to just jump right into your unique mechanics and ideas?
Well look no further! This Unity project contains all the most annoying basic project setup tasks ready to go, plus several hand and cool utility, functionality, or flair elements!

## Current Contents of project

### UI
- Main menu that leads to main scene
- Pause screen in main scene that can quit back to main menu
- Simple script that makes UI elements pop out when hovered over (mostly used for button clarity)

### Player Mechanics
- Basic player movement and functionality:
  - Seperate player prefabs for a player controlled by the Unity Character Controller and Rigidbody physics
  - Seperate player prefabs for first person and third person camera angles
